Acordeon

permite mostrar y ocultar contenido de manera vertical, en forma de seccion (plegables), sin bordes exteriores ni espacios entre las tarjetas del acordeón. Su estilo "flush" hace que el contenido se vea más limpio y se integre mejor con el fondo o el diseño contenedor.

This is the first item’s accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It’s also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.

breadcrumb

El breadcrumb (miga de pan) en Bootstrap es un componente de navegación que muestra la ruta o ubicación actual del usuario dentro de un sitio web. Sirve para que las personas sepan en qué página están y puedan volver fácilmente a páginas anteriores.

Alertas

Un alert en Bootstrap es un mensaje visual que sirve para llamar la atención del usuario sobre información importante, como advertencias, errores, éxitos o avisos informativos. Cada alerta tiene un color diferente según el tipo de mensaje que representa (ejemplo: rojo para errores, verde para éxito, amarillo para advertencias, etc.).
